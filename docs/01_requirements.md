# Day1–3 ITオンボーディングアシスタント 要件定義
## アプリ概要
- 目的:
入社直後（Day1–3）のIT初期設定・トラブル対応を自己解決可能にし、教育担当者の繰り返しサポート工数を削減する。
- ターゲット:
新入社員（Windows + iPhone配布環境）
および1対1でサポートする教育担当者
- 規模:
個人開発（Next.js + Tailwind）
想定ユーザー数：〜1000人（PoC対象は約20人）

## MVP機能
### Day別チェックリスト表示（Day1–3）
- Day1：ログイン・M365・Teams・iPhone初期設定
- Day2：ポータル・プリンタ・VPN・セキュリティ確認
- Day3：よくあるトラブル対応
- 各項目に「結論 → 手順 → 注意点」テンプレ表示
### 進捗管理・完了記録機能
- 各項目に「解決した / 解決しなかった」ボタン
- ユーザーごとの進捗表示（完了数／全体数）
- 未完了のみ表示切替
### 教育担当への導線機能
- 「この手順を送る」リンクコピー機能
- 未解決時の問い合わせ導線（Teamsチャットまたはフォーム）
- 利用ログ取得（閲覧・解決・未解決・問い合わせクリック）

## 将来機能
- 検索型FAQエージェント連携
- 自然文入力によるカテゴリ分類
- Day1–3以外のFAQ拡張
- ログ分析ダッシュボード
- 未解決ランキング表示
- よくある質問Top表示
- 教育担当向け改善レポート
- RAG連携・マニュアル統合
- 既存Wordマニュアルの自動変換・取り込み
- 関連FAQ自動提示
